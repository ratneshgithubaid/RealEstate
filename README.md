# Real Estate Website (React + Vite + Tailwind)

A modern real estate UI built with React, Vite, and Tailwind CSS.

## Features
- Fully responsive UI
- Property listing sections
- Smooth animations
- Clean code structure

## Tech Stack
- React
- Vite
- Tailwind CSS

## Run Locally

npm install
npm run dev

## Build Command
npm run build
